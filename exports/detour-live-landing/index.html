Capabilities

Everything agents need to ship

Detour gives hosted agents a shared workspace with persistent state, tool access, and cross-agent coordination.

🎯
Agent Orchestration
Detour acts as the host agent, spawning and coordinating Codex, Claude, and browser subagents with explicit task delegation and result merging.
⚡
Codex Code Engine
OpenAI Codex subagent handles diffs, type-checking, test execution, and iterative fixes inside a sandboxed workspace clone.
✦
Claude Reasoning
Claude subagent handles planning, design decisions, and prose — long-context analysis with citations fed back to Codex for implementation.
🖥
Live Terminal
Every agent action streams into a shared terminal. Watch pnpm, tsc, and test runners execute in real time with colorized, searchable output.
🔀
Diff + Code Pane
Inline diffs appear as agents write code — green additions, red removals, line numbers, and syntax highlighting across all touched files.
👁
Preview Renderer
UI components and API responses render live in the preview pane. Browser agent screenshots are inlined alongside terminal output.
