## Layers & ownership
- **View**: `src/main/**` (React webview process) ‚Äî talks to bun ONLY via `src/shared/rpc` + `src/main/rpc.ts`.
- **Contract**: `src/shared/**` ‚Äî wire types + RPC schema (single source of truth; must be a leaf).
- **RPC boundary**: `src/bun/core/rpc/handlers/**` ‚Äî adapt transport ‚Üí core services.
- **Application/services**: `src/bun/core/**` (services + the eliza AgentRuntime; composition root is `runtime.ts`, boot wiring in `index.ts`).
- **Features**: `src/bun/features/**` ‚Äî kernel-registered modules (window/hub opening).
- **Kernel**: `src/bun/kernel/**` ‚Äî tray / windows / events / view-url (Core ‚Üî window bridge).
- **Adapters**: `src/bun/plugins/**` ‚Äî Detour/eliza plugins.
- **Vendored (excluded from checks)**: `eliza/**`.

## Canonical lane
```
view (src/main) -> src/shared/rpc -> src/bun/core/rpc/handlers -> src/bun/core service -> src/bun/plugins adapter
```
Kernel features open windows/hub via the event bus; never let a view reach into `src/bun` except through `src/shared`.

## Window-open dispatch (single source of truth)
`src/shared/window-targets.ts`: `WINDOW_OPEN_MESSAGE` / `WINDOW_OPEN_KERNEL_EVENT`.
Do NOT re-derive `uiOpen*` names anywhere else (the `windowOpen` RPC, the `detour://` url-scheme, and the kernel bridge all read these maps).
